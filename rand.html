O Rand Sul-Africano

O Rand é a moeda oficial da Republica Da Africa Do Sul e é usado oficialmente em: Lesoto, Namíbia e Eswatini.

    O Rand também tem 10 nomes diferentes:

    Rand Suid-Afrikaanse ( Afrikaans )
    iRanti yeSewula Afrika ( sul de Ndebele )
    iRanti yoMzantsi Afrika ( Xhosa )
    iRandi laseNingizimu Afrika ( Zulu )
    liRandi laseNingizimu Afrika ( suazi )
    Ranta ya Afrika-Borwa ( Soto do Norte )
    Ranta ya Afrika Borwa ( Sotho )
    Ranta ya Aforika Borwa ( Tswana )
    Rhandi ya Afrika-Dzonga ( Tsonga )
    Rannda ya Afurika Tshipembe ( Venda )
